
・start emacs at terminal: $ emacs -nw filename.txt

・open multiple files: cntl(C) + F
・switch files: C + X  ← →
・close one of files: C + X  K

・select: shift + ↑ ↓ ← →

・copy: alt(M) + w

・cut: left: C + w

・paste: C + y

・comment, uncomment: M + ";(+)"

・delete the line: C + k

・save: C + x   C + save

・save as another file: C + x  C + w

・end emacs: C + x  C + c
