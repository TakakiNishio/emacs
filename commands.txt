M = alt
C = left ctrl

・start emacs in terminal: $ emacs -nw filename.txt

・open multiple files: C + f

・switch files: C + x  ← →

・close one of files: C + x  k

・select: shift + ↑ ↓ ← →

・copy: M + w

・cut: C + w

・paste: C + y

・scroll down: C + v

・scroll up: M + v

・repcace: M + shift + 5(%) then "words" then space

・cut: left: C + w

・paste: C + y

・comment, uncomment: M + ;(+)

・delete the line: C + k

・save: C + x   C + s

・save as another file: C + x  C + w

・end emacs: C + x  C + c
